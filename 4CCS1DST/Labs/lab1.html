a.       Add the following two constructors to this class:

Test these constructors by adding appropriate instructions to the main method.

    1. To check if two objects represent the same data, we do not use the “==“ operator, but an appropriately defined method equals:

public boolean equals(Object obj) { … }

Each class either explicitly defines this method, or inherits it from its superclass, or from class Object (which is the base superclass for all classes). For example, in

Integer i = new Integer(5);
Integer j = new Integer(5);
if ( i == j ) { ... }
if ( i.equals(j) ) { ... }

objects i and j represent the same data (integer number 5) but are physically different objects, so the first condition “i == j”, which checks if  i and j refer to the same object, is false. But objects i and j represent the same data, so the second condition “i.equals(j)” is true.

The method equals defined in class Object returns true, if and only if, the two reference variables refer to the same object. Add the following instructions to the main method in class Pair to test the equals method inherited from class Object:

 

Pair<String, Integer> pairA = new Pair<String, Integer>("abc", 1);

Pair<String, Integer> pairAA = pairA;

Pair<String, Integer> pairB = new Pair<String, Integer>("abc", 2);

Pair<String, Integer> pairC = new Pair<String, Integer>("xyz", 1);

Pair<String, Integer> pairD = new Pair<String, Integer>("abc", 1);

Pair<String, Integer> pairE = new Pair<String, Integer>();

Pair<String, Integer> pairF = new Pair<String, Integer>(null, 1);

Pair<String, Integer> pairG = null;


System.out.println( pairA.equals(pairAA) );
System.out.println( pairA.equals(pairB) );
System.out.println( pairA.equals(pairC) );
System.out.println( pairA.equals(pairD) );
System.out.println( pairA.equals(null) );
System.out.println( pairA.equals(new Integer(5)) );

System.out.println( pairE.equals(pairA) );

System.out.println( pairF.equals(pairA) );

System.out.println( pairA.equals(pairG) );

 

Run the class Pair with this test. Is the output as you expected?
